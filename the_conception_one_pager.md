**THE CONCEPTION**

Game Design One Pager · v0.1

**CORE DEFINITION**

A God-view RTS where **intelligence is your most scarce resource.** You
can\'t see what the enemy is building without actively sending scouts.

+-----------------------------------------------------------------------+
| **THE FEELING --- WHY THIS GAME EXISTS**                              |
|                                                                       |
| The player watches a border they thought was secure collapse in real  |
| time. Not because they lacked firepower --- because they stopped      |
| asking questions.                                                     |
|                                                                       |
| **That feeling is the entire game. Every mechanic exists to create    |
| it.**                                                                 |

**THE ONE MECHANIC THAT MAKES THIS DIFFERENT**

**Live Fog of War.** Fog does not lift passively as you explore. Tiles
you leave go dark again. The enemy can build, expand, and mass forces
completely invisible to you --- unless you commit scouts continuously.

Every scout is a decision: spend resources to know, or save resources
and guess.

  ---------------- --------------- ------------------- -------------------
  **Unit**         **Cost**        **Sight Radius**    **Risk**

  Scout (foot)     50 res / 8s     4 tiles             Can be killed ---
                                                       info lost

  Scout Vehicle    150 res / 15s   8 tiles             Visible to enemy,
                                                       can be ambushed

  Drone            300 res /       Full sector         Can be shot down
                   cooldown                            --- expensive loss
